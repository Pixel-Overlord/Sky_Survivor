# Sky Survivor

Sky Survivor is a 3D endless arcade game built in Unity.

## Features

- ✈️ Endless plane survival gameplay
- 🏙️ Procedural city generation
- ♻️ Object Pooling
- 🧱 Randomized building spawning
- 🎮 Arcade flight controls
- 💥 Game Over system
- 📈 Endless score system

## Tech Stack

- Unity 2022.3 LTS
- C#
- Object Pooling
- Procedural Generation
- OOP
- Unity Physics
